# 程序设计

## 指令

```
MOV X,1			//x=1

XCHGH X,Y		//交换XY值 

XLAT TABLE或XLAT //把待查表格的一个字节内容送到AL累加器，TABLE为待查表格首地址，用于转码

堆栈操作
PUSH X	//入栈
POP  X  //出栈

标志传送：
地址传送：

算术指令
ADD X,Y  //X=X+Y
ADC X,Y  //X=X+Y+CF,带进位加法指令
INC X 	 //X+=1

SUB X,Y	 //X=X-Y
SBB X,Y	 //X=X-Y-CF,带借位减法
DEC X	 //X-=1

NEG X 	 //X=0-X,取补
CMP X,Y	 //比较指令，改变标志位

MUL X	 //寄存器*X
IMUL X	 //带符号乘法
DIV X	 //寄存器/X
IDIV X	 //带符号除法
CBW 	 //字节扩展指令，字节转换字，字转换双字
CWD		 //字扩展，字转为双字长

十进制调整指令


逻辑操作指令
AND OR NOT XOR(异或) TEST
逻辑移位指令
循环移位指令

程序控制指令
JMP X	//无条件跳转指令
JCC X	//条件跳转
LOOP 标号 //循环指令
LOOPZ/LOOPE 标号	//条件循环
过程调用以及返回指令和中断指令
```
